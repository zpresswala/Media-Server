Main parameters

Parameter Type Req Descriptions
apiURL string

The api base url

Default: https://www.voicebase.com/services

Example: 
apiURL: 'https://beta.voicebase.com/services'
apiKey string

your voicebase api key

Example: 
apiKey: 'ABCDEFGH-1234-ABCD-1234-1A2B3C4D5F6G'
password string

your voicebase api password

Example: 
password: 'you_vb_password'
token string

is indicated instead of apikey and password

Example: 
token: 'db250aaf...e581a'
example string

Used only in VoiceBase example files

Default: false

Example: 
example: true
width integer

Custom width for widget block

Default: 640

Example: 
width: 300
result_block_id string

ID of the alternative block for showing results

Default: searchresultblock

Example: 
result_block_id: 'customId'

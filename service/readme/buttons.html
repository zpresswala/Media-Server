Buttons

actionFlag - it is used for turning on/off the buttons

Buttons list (hover over to see example):

  • downloadMedia [true]
  • remove [true]
  • favorite [true]
  • help [true]
  • fullscreen [true]
  • share [true] * additional setup is required
  • evernote [true] * additional setup is required
  • prev [true]
  • next [true]
  • downloadTranscript [true]
  • edit [true]
  • print [true]
  • readermode [true]
  • orderTranscript [true]

Example script:


/* init VoiceBase plugin settings */
jQuery('#player').voicebase({
    ...
    actionFlag: {
        remove: false,
        comment: false
    }
});

Additional parameters:

Share

To have Share function accessible you need to add addThis library:

<script type="text/javascript"
                        src="//s7.addthis.com/js/300/addthis_widget.js#pubid=ra-51ae6dc64168357f"></script>

Evernote

To have export function to Evernote accessible you need to add filepicker library and install API Key:

<script type="text/javascript" src="//api.filepicker.io/v1/filepicker.js"></script>
    <script type="text/javascript">
    filepicker.setKey('YourFilepickerApiKey');
    </script>

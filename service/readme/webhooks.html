WebHooks - interception of default events of buttons

Supported webhooks:

  • comment
  • remove
  • favoriteTrue
  • favoriteFalse
  • help
  • evernote
  • editTranscript
  • print
  • downloadMedia
  • downloadTranscript
  • socialShare
  • vbShare
  • orderTranscript
  • removeTopic
  • removeKeyword
  • keywordUp
  • keywordDown
  • keywordFirst
  • commentEdit
  • commentDelete
  • readermode
  • readermodeExit
  • fullscreen
  • fullscreenExit

Example script:


function myCustomFunction() {
    alert(123);
}
/* init player settings */
jQuery('#player').voicebase({
    ...
    webHooks: {
        socialShare: function(){
        myCustomFunction();
    },
    orderTranscript: function(){
        myCustomFunction();
    }
});
